ConvoConnect - ChatApp

ConvoConnect is a real-time chat application where users can create their profiles and chat with other users. It supports both incoming and ongoing messages, with all messages being saved to a database and fetched for display. The app uses PHP, MySQL, and AJAX to create a seamless real-time messaging experience, allowing users to interact with each other without page reloads.

Quran App using API

This app is built using Laravel, utilizing the Quran API to fetch 30 Juzs (sections of the Quran) and 114 Surahs (chapters) along with their audio. Users can browse through the Quranic content and listen to the audio of each Surah and Juz. The app provides a seamless experience for Quranic studies and audio playback, all built with Laravel for optimal performance.
